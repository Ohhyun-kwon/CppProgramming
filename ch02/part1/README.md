## Q1. 전처리 또는 선행처리 단계를 설명하라.
A1. 편집 -> 컴파일 -> 링크 -> 실행 과정 중 컴파일 이전에 처리되는 단계이며, #으로 시작하는 전처리 지시문을 처리한다. 

## Q2. IDE의 종류에 대해 조사하시오.  
A2. Visual Studio 는 ms에서 개발하고 C 계열 개발에 많이 쓴다. Eclipse 는 Java 개발에 사용되는 오픈소스 IDE이다. 이외에도 다양한 IDE가 존재한다.

## Q3. IDE에 포함된 기능은 무엇인가?  
A3. 코드 편집, 컴파일러, 디버거, 빌드 등등의 기능을 포함하고 있다.
